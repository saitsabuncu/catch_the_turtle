## ğŸ How It Works

The game uses the **turtle** module to create an interactive screen:
- The turtle randomly moves to different positions on the screen.
- The `onclick` event tracks when the turtle is clicked and updates the score.
- A timer is implemented using the `time` module to limit the gameplay to 30 seconds.
- At the end of the game, the final score is displayed.
